Clínicamente: Neuropsicología
=============================

Descripción:
------------
Este proyecto es un sitio web para una clínica neuropsicológica. Incluye información sobre la clínica, sus ubicaciones y una integración con Google Maps.

Estructura del Proyecto:
------------------------
- index.html: Archivo principal que contiene la estructura HTML del sitio web.
- estilos.css: Hoja de estilos que define la apariencia visual del sitio.
- README.txt: Documento que describe el proyecto y su contenido.

Instrucciones:
--------------
1. Asegúrate de tener los tres archivos en la misma carpeta.
2. Reemplaza "TU_API_KEY" en el script de Google Maps por tu clave de API válida.
3. Coloca las imágenes necesarias (logo, fondo) en la misma carpeta.
4. Abre el archivo `index.html` en tu navegador para ver el sitio web.
5. Modifica los contenidos según sea necesario para adaptarlos a tus necesidades.

Créditos:
---------
Diseño creado con base en principios modernos de diseño web y la paleta de colores del logo proporcionado.
